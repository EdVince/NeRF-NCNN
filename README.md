# NeRF-NCNN

NeRF implemented by [ncnn](https://github.com/Tencent/ncnn) framework based on C++, reference from [taichi-nerfs](https://github.com/taichi-dev/taichi-nerfs)

***Performance***
|                  | taichi & python | ncnn & c++ |
| ---------------- | --------------- | ---------- |
| i7-12700+RTX3060 | ~15fps          | ~0.6fps    |
